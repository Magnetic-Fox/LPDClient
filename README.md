# LPD Client

The simplest possible LPD client being in fact a one shot procedure for queueing a print job.  
Main code was suggested by Google AI (as I wasn't able to find any descriptions on the LPD internet protocol, as RFC is about local system and not the actual internet protocol).

## How to use it?

It's as simple as it can be. Just pass data, IP (or domain address) and optionally port (if not using default 515) and queue name to the `lpd_print` function.
